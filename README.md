# 🛒 Project Elec Shop
**Студент:** Бексұлтан

### 🚀 Іске қосу алгоритмі:
1. Терминал 1: `node server.cjs` (Backend іске қосу)
2. Терминал 2: `npm run dev` (Frontend іске қосу)

### 💡 Негізгі функциялар:
- PostgreSQL-ден тауарлар тізімін алу.
- React Router арқылы беттер арасында ауысу.
- Себетке тауар қосу/өшіру және сомманы есептеу.
